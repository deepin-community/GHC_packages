=head1 NAME

dh_haskell_buildinst - build and install Cabalized libraries

=head1 SYNOPSIS

B<dh_haskell_buildinst> [S<I<debhelper options>>]

=head1 DESCRIPTION

dh_haskell_buildinst is a debhelper program that helps with preparing Haskell
libraries.

It automates building libraries for the different supported Haskell systems in
Debian.

=head1 BUGS

hugs and ghc6 are the only supported targets at the moment.  Cabal does
not yet support nhc98.
